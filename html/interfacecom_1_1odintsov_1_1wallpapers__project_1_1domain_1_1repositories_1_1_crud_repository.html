Detailed Description

A generic base repository interface providing standard CRUD operations and advanced filtering capabilities.

  • Parameters
    <T>The entity type managed by the repository.
    <ID>The type of the entity's identifier (usually String or UUID).
    <F>The filter DTO type used for faceted search and criteria.
