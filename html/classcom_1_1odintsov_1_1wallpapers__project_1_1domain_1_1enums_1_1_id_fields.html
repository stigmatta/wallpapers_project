Detailed Description

Utility class containing database column name constants for Foreign Keys.

This class centralizes the naming of ID references used in @JoinColumn annotations. By standardizing these names, the system ensures that relationship mappings across different entities and join tables (e.g., Many-to-Many links) remain consistent and maintainable.
