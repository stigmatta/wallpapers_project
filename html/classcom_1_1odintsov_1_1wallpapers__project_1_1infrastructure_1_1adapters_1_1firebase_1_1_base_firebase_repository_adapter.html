Detailed Description

Generic Firestore adapter providing a bridge between the Domain Repository and Google Cloud NoSQL storage.

This class handles the complexity of Firestore document mapping, asynchronous future resolution, and paginated filtering.

  • Parameters
    <T>The Domain Entity type.
    <ID>The ID type (mapped to Document ID).
    <F>The Filter DTO type for dynamic queries.
