Detailed Description

Firestore adapter for UserSession persistence.

This implementation uses the Google Cloud Firestore SDK to manage session data. It performs manual mapping of domain entities to Firestore maps to ensure specific formatting for date fields (ISO_LOCAL_DATE_TIME).
