Detailed Description

Abstract base implementation of BaseService providing standard CRUD operations.

This class orchestrates the interaction between a CrudRepository and a DtoMapper. It handles common business logic such as checking for entity existence before updates/deletions and throwing EntityNotFoundException when lookups fail.

Parameters
<T>The domain entity type
<ID>The identifier type (e.g., Long, UUID)
<FilterDTO>The DTO used for filtering search results
<ListResponse>The DTO for lightweight list displays
<DetailedResponse>The DTO for full entity details
<R>The specific repository type extending CrudRepository
