Detailed Description

Functional interface for applying domain-specific filters to a Firestore Query.

Implementations of this interface translate Domain Filter DTOs into Firestore-specific query constraints like whereEqualTo, whereArrayContains, or range filters.

<F>The Type of the Filter DTO.
