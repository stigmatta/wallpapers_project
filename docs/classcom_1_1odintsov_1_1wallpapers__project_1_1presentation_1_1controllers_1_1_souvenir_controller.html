Detailed Description

REST Controller providing endpoints for souvenir product management.

This controller handles public requests to browse the souvenir catalog, providing support for dynamic filtering via SouvenirFilter and pagination. It supports resource retrieval by both unique UUID and SEO-friendly slugs.
